Voice of Emotions - Emotion Recognition using audio dataÂ¶

Business Objective / ApplicationÂ¶

  • Augmenting Human judgment: example: Lie detection
  • Annotating Customer Service Calls: Clear Indication of whether customer is satisfied with the service he / she received over phone.

Database - IEMOCAP Interactive Emotional Motion Capture DatabaseÂ¶

Detailed Description

Key Points

  • Session consist of a male and female actors acting improvised scripts. Data Contains 5 such Session [Conversation between 5 Male / Female Actors]
  • Each session contains 7 improvisied scripts emoting 5 different emotions:
    • Frustation, Sadness, Happiness, Anger and Neutral
  • Each Sessions audio is seperated into several chunks and carefully annotated with different emotions.
  • 5 Evaluvators categorized Each chunk of actual audio into above 5 categories. Average of that is used to describe emotions related to that chunk.

Trying to AnswerÂ¶

  • How effective can we classify emotions from voice /audio data. [Emotions are conveyed through several channels - Text, Facial features, head / hand motion apart from voice]

Planned ApprochÂ¶

  • Feature extraction and prepration - Looking at features like MFCC, Pitch, Energy and respective statistics.
  • Classification using xgboost classifier

Machine Learning PipelineÂ¶

  • Feature Extraction from audio
    • MFCC, Pitch, Energy and respective statistics.
  • Feature scaling using standard scalar
  • Label encoding using scikit learn label encoder
  • Test train split
  • XG Boost classifier
  • Label decoder
  • Evaluation metric
    • Accuracy, Class confusion matrix, F-score and ROC AUC

def plot_confusion_matrix(cm, classes,
                          normalize=False,
                          title='Confusion matrix',
                          cmap=plt.cm.Blues):
    """
    This function prints and plots the confusion matrix.
    Normalization can be applied by setting `normalize=True`.
    """
    # plt.figure(figsize=(8,8))
    plt.imshow(cm, interpolation='nearest', cmap=cmap)
    plt.title(title)
    plt.colorbar()
    tick_marks = np.arange(len(classes))
    plt.xticks(tick_marks, classes, rotation=45)
    plt.yticks(tick_marks, classes)

    if normalize:
        cm = cm.astype('float') / cm.sum(axis=1)[:, np.newaxis]
        print("Normalized confusion matrix")
    else:
        print('Confusion matrix, without normalization')

    print(cm)

    thresh = cm.max() / 2.
    for i, j in itertools.product(range(cm.shape[0]), range(cm.shape[1])):
        plt.text(j, i, cm[i, j],
                 horizontalalignment="center",
                 color="white" if cm[i, j] > thresh else "black")

    plt.tight_layout()
    plt.ylabel('True label')
    plt.xlabel('Predicted label')
    
def one_hot_encoder(true_labels, num_records, num_classes):
    temp = np.array(true_labels[:num_records])
    true_labels = np.zeros((num_records, num_classes))
    true_labels[np.arange(num_records), temp] = 1
    return true_labels

def display_results(y_test, pred_probs, cm = True):
    pred = np.argmax(pred_probs, axis=-1)
    one_hot_true = one_hot_encoder(y_test, len(pred), len(label_dict))
    print('Test Set Accuracy =  {0:.2f}'.format(accuracy_score(y_test, pred)))
    print('Test Set F-score =  {0:.2f}'.format(f1_score(y_test, pred, average='macro')))
    print('ROC AUC = {0:.3f}'.format(roc_auc_score(y_true=one_hot_true, y_score=pred_probs, average='macro')))
    if cm:
        plot_confusion_matrix(confusion_matrix(y_test, pred), classes=label_dict.keys())
